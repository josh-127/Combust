Portable Toolchain (PTC)
========================

Toolchain for native application development. Currently includes a C compiler.

Building
--------

### Requirements
| Dependency       | Version  |
|------------------|----------|
| CentOS           | 7.5.1804 |
| GNU Make         | 3.82     |
| GNU GCC          | 4.8.5    |

### Installing Requirements
```
yum install autoconf
yum install automake
yum install make
yum install gcc
```

### Building PTC
```
make
```
